What is neural network?

Neural network is computer graph. In a graph, every node represent a Tensor / Value / Matrix / Scalar. An edge is function argument. A node with an incoming edge is a function of this edge's tail node. A function can be nullary, unary, binary, ..., n-ary which means it takes 0 to N parameters. Comparaed to static graph, dynamic graph is more flexible but more slower.
